# PET Saúde Digital - UNCISAL

Repositório destinado ao registro e desenvolvimento de atividades, pesquisas e soluções tecnológicas produzidas para o programa *PET Saúde Digital*.

### 📋 Sobre o Programa:
O PET Saúde Digital é uma iniciativa que une educação pelo trabalho e inovação, focando na aplicação de tecnologias da informação para a melhoria dos processos de saúde.

### 🛠 Tecnologias e Estudos:
* Desenvolvimento de Sistemas para Internet.
* Inovação em Saúde Digital.
* Metodologias Ágeis e Colaboração.

### 🚀 Objetivo:
Documentar o progresso técnico e as contribuições desenvolvidas durante a minha participação no programa, aplicando conhecimentos de *Sistemas para Internet* em desafios reais da área da saúde.
